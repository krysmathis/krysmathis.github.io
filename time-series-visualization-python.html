Loading and Visualizing Time Series Plot¶

Getting the time part right¶

It is recommended that when working with datetime in python that the datetimes be in the format of datetime64. To check that your dataframe has the correct data types, print the dtypes attribute:

print(df.dtypes)

To convert a column which is imported as object to a datetime, you can use the pandas method to_datetime()

df['date'] = pd.to_datetime(df.date)

In general, set the datetime column as the index because, beyond other reasons, it makes plotting simpler¶

Plot Formatting¶

Seasonality, Trend and Noise¶

Autocorrelation and Partial Autocorrelation¶

  • Autocorrelation is measured as the correlation between a time series and a delayed copy of itself
  • It is used to find repetitive patterns or periodic signal in time series
from statsmodels.graphics import tsaplots

# Display the autocorrelation plot of your time series
fig = tsaplots.plot_acf(co2_levels['co2'], lags=24)

Autocorrelation values close to 0 indicate no correlation, but if they are close to 1 or -1 then there is a strong positive or negative correlation

from statsmodels.graphics import tsaplots

# Display the partial autocorrelation plot of your time series
fig = tsaplots.plot_pacf(co2_levels['co2'], lags=24)

# Show plot
plt.show()

If partial autocorrelation values are beyond this confidence interval regions, then you can assume that the observed partial autocorrelation values are statistically significant.

Working with more than one time series¶

Generally not a problem if they are in the same dataframe and share and index.

One issue that comes up is the display within a plot

The colormap argument helps to avoid a situation where the same color is used for multiple datapoints

ax = df.plot(colormap='Dark2', figsize=(14,7))

Enhancing your plot with information

Using a clustermap instead¶

# Import seaborn library
import seaborn as sns

# Get correlation matrix of the meat DataFrame
corr_meat = meat.corr(method='pearson')

# Customize the heatmap of the corr_meat correlation matrix and rotate the x-axis labels
fig = sns.clustermap(corr_meat,
                     row_cluster=True,
                     col_cluster=True,
                     figsize=(10, 10))

plt.setp(fig.ax_heatmap.xaxis.get_majorticklabels(), rotation=90)
plt.setp(fig.ax_heatmap.yaxis.get_majorticklabels(), rotation=0)
plt.show()
